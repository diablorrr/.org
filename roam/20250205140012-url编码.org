* url编码
- 简介 :: 特殊字符 => url安全格式 => 避免特殊字符解析错误、误解析
** url编码规则
- 直接保留，不编码
  1. 字母和数字（A-Z, a-z, 0-9）
  2. 安全字符（- _ . ~）
- %十六进制
  1. 特殊字符（如空格、#、?、&）例如：空格 ( ) → %20；? → %3F；& → %26
  2. 非 ASCII 字符（如中文）例如："你好" → "%E4%BD%A0%E5%A5%BD"
