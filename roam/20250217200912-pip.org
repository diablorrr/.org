* 使用
** 查看pip使用的配置文件
#+begin_src bash
pip config list --verbose
#+end_src

** 解决timm安装失败问题
#+begin_src bash
pip install timm          # 失败
conda install timm        # 失败
pip install timm          # 加镜像源 失败
pip install timm==0.4.12  # 成功 速度很快
#+end_src
