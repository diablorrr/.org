* artist-mode
1. emacs 的绘图模式，用于绘制 ASCII 图， *常用操作* 如下
   | 按键           | 作用             |
   |----------------+------------------|
   | 鼠标中键       | 切换画笔         |
   | 鼠标左键       | 使用画笔         |
   | 鼠标左键+shift | 使用画笔特殊功能 |
   | 鼠标右键       | 使用橡皮擦       |
   | 鼠标右键+shift | 使用范围橡皮擦   |
