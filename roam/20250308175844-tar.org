#+title: tar

* 使用
** 解压.tar.gz文件
1. tar -zxvf file.tar.gz



* 查询
| 参数 | 作用                               |
|------+------------------------------------|
| -x   | 解压文件                           |
| -v   | 显示解压详细信息                   |
| -f   | 指定操作的文件名(必须放最后)       |
| -z   | 使用gzip压缩/解压 :: .tar.gz或.tgz |
| -j   | 使用bzip2压缩/解压 :: .tar.bz2     |
| -J   | 使用xz压缩/解压 :: .tar.xz         |
** gzip、bzip2、xz对比
| 压缩方式 | 速度 | 压缩率 |
|----------+------+--------|
| gzip     | 快   | 低     |
| bzip2    | 中   | 中     |
| xz       | 低   | 高     |
