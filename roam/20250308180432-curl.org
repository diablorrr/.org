#+title: curl


* 查询
| 命令                                | 作用                  |
|-------------------------------------+-----------------------|
| curl https://google.com             | 发送GET请求           |
| curl -I https://google.com          | 仅获取响应头          |
| curl -v https://google.com          | 显示详细请求/响应过程 |
| curl -X <METHOD> https://google.com | 指定请求方法，如:POST |


* 概念
- 简介 ::
  1. 网络请求工具
  2. 支持 HTTP、HTTPS、FTP、SFTP、SCP、LDAP 等多种协议
