* 使用
#+begin_src bash
curl https://example.com    # GET请求获取网页内容
curl -i https://example.com # 显示响应头
curl -v https://example.com # 详细模式：显示请求头、响应头、调试信息
#+end_src


* 概念
- 简介 ::
  1. 网络请求工具
  2. 支持 HTTP、HTTPS、FTP、SFTP、SCP、LDAP 等多种协议
