#+title: Dockerfile

* 查询
| 命令                         | 作用                                              |
|------------------------------+---------------------------------------------------|
| FROM                         | 指定基础镜像                                      |
| WORKDIR                      | 设置工作目录(若不存在自动创建)                    |
| RUN                          | 执行命令，并创建新的镜像层                        |
| COPY <宿主机路径> <容器路径> | 复制文件                                          |
| CMD                          | 指定容器创建时的默认命令(可以覆盖)                |
| SHELL                        | 覆盖Docker中默认的shell，用于RUN、CMD、ENTRYPOINT |



* 概念
- 简介 ::
  1. 包含一系列指令的文本文件
  2. 用于自动化构建[[id:a47cc941-ccc6-4893-9862-d5b245ea912b][docker]]镜像
