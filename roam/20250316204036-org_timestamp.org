* 查询
|----------+------------------------+----------------------------------------|
|          | 函数                   | 作用                                   |
|----------+------------------------+----------------------------------------|
| 时间戳   | (C-c .) org-timestamp  | 插入激活时间                           |
|          | org-timestamp-inactive | 插入非激活时间                         |
|----------+------------------------+----------------------------------------|
| 重复任务 | +1d                    | 多次任务，需完成同样完成。所有任务done |
|          | .+1d                   | 多次任务，完成一次。所有任务done       |
|----------+------------------------+----------------------------------------|



* 概念
- 简介 ::
  + 激活时间<> :: 加入日程表
  + 非激活时间[] :: 不加入日程表
  #+begin_example
  <2024-12-22 日 +1d>    # 激活时间
  [2024-12-22 日 17:07]  # 非激活时间
  #+end_example
