#+title: 内存对齐

* 概念
- 简介 ::
  1. 用于提高cpu访问内存的效率 => cpu以块的形式访问内存，若内存未对齐，cpu需要多次内存访问获取完整数据，并进行额外的位移和拼接操作 => 降低效率
  2. 对齐原则：
     1) 结构体内除第一个成员外，其他成员所在的地址是数据类型所占字节的整数倍 => 如果不是整数倍 => 编译器插入填充字节保证成员对齐
     2) 结构体内成员各自对齐后，结构体本身还要进行一次内存对齐 => 保证结构体大小是最大成员的整数倍
