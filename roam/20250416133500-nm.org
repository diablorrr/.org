#+title: nm

* 查询
** 命令
| 参数 | 作用                       |
|------+----------------------------|
| -C   | 解码[[id:8ab4df56-e11f-42b8-87f8-4daa2fd045db][cpp]]符号，按地址排序    |
| -u   | 查看目标文件中未定义的符号 |
| -A   | 显示每个符号的文件名       |
** 输出符号类型
| 符号 | 作用                                  |
|------+---------------------------------------|
| T、t | 代码段(Text)符号 (大写全局，小写局部) |
| D、d | 数据段(Data)符号                      |
| B、b | BSS段符号 (未初始化的全局变量)        |
| U    | 未定义符号 (需要外部链接)             |
| R、r | 只读数据段符号                        |
| C    | 公共符号                              |
| W、w | 弱引用符号                            |

- 简介 ::
  1. [[id:ec7aef91-2628-4ba9-b300-16652314877f][linux]]中的命令行工具
