#+title: ps


* 使用
** 显示所有进程的完整列表
1. ps -ef


* 查询
| 命令 | 选项 | 作用                           |
|------+------+--------------------------------|
| ps   | -e   | 显示所有进程                   |
|      | -f   | 显示完整格式(PPID、启动时间等) |
|      | -H   | 进程树层级(进程父子关系)       |


* 概念
- 简介 :: [[id:ec7aef91-2628-4ba9-b300-16652314877f][linux]]的命令行工具，用于显示正在运行的进程的状态
