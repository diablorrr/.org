#+title: sed

* 查询
- 语法 :: sed <选项> '<脚本>' <文件名>
** 选项
| 选项 | 作用          | 示例 |
|------+---------------+------|
| -n   | (需配合p使用) |      |
| -i   | 直接修改文件  |      |
** 脚本
| 脚本                        | 参数   | 作用                     |
|-----------------------------+--------+--------------------------|
| sed 's/old/new/<参数>' file |        | 替换文本                 |
|                             | <数字> | 替换第<数字>个匹配的文本 |
|                             | g      | 全局替换                 |
|                             | i      | 忽略大小写               |
