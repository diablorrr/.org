* git submodule update时更新部分子模块
1. [[id:e396235e-8b41-475c-8fe2-40d3def9f9b2][.gitmodules]] => 注释掉不需要的子模块

创建新git仓库
touch .gitmodules
git submodule <子模块仓库URL> [可选子模块目录名]
