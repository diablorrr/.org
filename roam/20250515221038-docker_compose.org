#+title: docker-compose

* 查询
| 类别      | 参数                    | 作用                           |
|-----------+-------------------------+--------------------------------|
| 启动/停止 | up -d                   | 启动所有服务(后台运行)         |
|           | down [-v]               | 停止并删除所有容器、网络、[卷] |
|-----------+-------------------------+--------------------------------|
| 管理服务  | start <服务名>          | 启动指定服务                   |
|           | stop <服务名>           | 停止指定服务                   |
|           | restart <服务名>        | 重启指定服务                   |
|-----------+-------------------------+--------------------------------|
| 状态查看  | ps [-a]                 | 查看运行状态                   |
|           | logs -f                 | 实时查看日志                   |
|-----------+-------------------------+--------------------------------|
| 调试      | exec <服务名> /bin/bash | 进入容器                       |
|           | config                  | 查看docker-compose.yml         |
