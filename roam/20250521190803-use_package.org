* use-package
| 块        | 作用                 |
|-----------+----------------------|
| :init     | 包加载前执行         |
| :config   | 包加载后执行         |
| :defer    | 直接延迟加载整个包   |
| :commands | 使用命令时，加载包   |
| :bind     | 使用快捷键时，加载包 |
